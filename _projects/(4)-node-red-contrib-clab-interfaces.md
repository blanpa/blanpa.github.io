# CompuLab IoT Gateway Interfaces for Node-RED

13 node categories enabling full hardware interaction with **CompuLab IoT Gateways** (IOT-GATE-iMX8, IOT-GATE-RPi, IOT-LINK, IOT-DIN-IMX8PLUS).

## Key Features

**I/O & Communication**
- GPIO digital input/output
- RS232 / RS485 serial communications
- CAN bus message handling
- Analog sensor inputs (4-20mA, 0-10V, PT100/PT1000)

**Device Control**
- System operations (RTC, watchdog, TPM, power management)
- Bluetooth adapter and device connectivity
- GPS positioning and satellite data

**Connectivity**
- Cellular/LTE modem management with signal monitoring
- WiFi scanning, connection, and access point modes
- Ethernet interface configuration
- Network diagnostics (ping, DNS, traceroute, port checks)

**Scalability**
Supports stacking up to 8 I/O expansion modules — enabling up to 64 analog channels, 64 digital inputs, and 64 digital outputs per gateway.
